HTML Tags
HTML Attributes
  1. HTML tags are used to hold the HTML element.
  1. HTML attributes are used to describe the characteristic of an HTML element in detail.
  1. HTML tag starts with < and ends with >
  1. HTML attributes are found only in the starting tag.
  1. HTML tags are almost like keywords where every single tag has unique meaning.
  1. HTML attributes specify various additional properties to the existing HTML element.
